This project is to store files used for VM deployment by the VMDB resource
https://vmdb.tower.am.health.ge.com

Files:
teams		: List of team names, format: "<team-lead-sso>|<short name>|<long team name>"
		  Note: short team name is alpha only and 2-8 characters long. SSO should be numeric only

Directories:
powercli	: Scripts used by powershell/powercli portion of pipelines
groovy		: Scripts used by groovy portion of pipelines
